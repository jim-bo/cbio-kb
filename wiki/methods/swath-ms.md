# SWATH-MS

## Overview

SWATH-MS (Sequential Window Acquisition of all THeoretical fragment ion spectra) is a data-independent acquisition (DIA) mass spectrometry approach for proteome quantification. Unlike data-dependent acquisition (DDA), SWATH-MS fragments all precursor ions within predefined m/z windows, enabling comprehensive and reproducible proteome-wide quantification. A sample-specific spectral library is first generated by DDA and used to extract peptide intensities from the SWATH data using tools such as OpenSWATH or Spectronaut.

## Used by

- Applied to 51/122 [HCC](../cancer_types/HCC.md) biopsies ([hcc_meric_2021](../datasets/hcc_meric_2021.md)) using Q Exactive Plus instrumentation, quantifying 6,167 proteins; spectral library built by MaxQuant with peak extraction by Spectronaut X; enabled integrated proteogenomic analysis of HCC subtypes and kinase activity [PMID:35508466](../papers/35508466.md)

## Notes

- SWATH-MS offers high reproducibility and proteome coverage, making it suitable for large clinical cohorts.
- Quantification is performed retrospectively using a spectral library, allowing re-analysis as libraries are updated.
- Commonly paired with phosphoproteomics (e.g., Fe(III)-IMAC enrichment) for integrated signaling analyses.
