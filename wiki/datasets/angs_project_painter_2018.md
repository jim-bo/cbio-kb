# Angiosarcoma Project (Painter et al., 2018)

## Overview

The Angiosarcoma Project is a patient-partnered rare-cancer genomics initiative curated by Painter et al. (2018) and available in cBioPortal under study ID `angs_project_painter_2018`. It uses a patient-driven enrollment model to collect tumor specimens and clinical data from angiosarcoma patients, enabling genomic profiling of this rare malignancy. The dataset is independent of single-center retrospective angiosarcoma cohorts.

## Composition

- Angiosarcoma ([ANGS](../cancer_types/ANGS.md)) tumors from a patient-partnered enrollment initiative.
- Includes primary and secondary (radiation-associated) angiosarcoma cases across multiple anatomical sites.
- Exact sample size and sub-cohort breakdown are available via cBioPortal.

## Assays / panels (linked)

- Genomic profiling assays (sequencing type and panel details available in the cBioPortal study).
